## Installation

### Required Files

This plugin requires these files to work in Obsidian:

| File | Description | Source |
|------|-------------|--------|
| `manifest.json` | Plugin metadata | **Create manually** (see below) |
| `main.js` | Compiled plugin code | Generated by `npm run build` |
| `styles.css` | Plugin styling | Already in repo |
| `versions.json` | Version compatibility | Already in repo |

### Setup Instructions

1. **Create manifest.json** (first time only):

```json
{
  "id": "ljoves-tools",
  "name": "LjoveS Tools",
  "version": "1.0.0",
  "minAppVersion": "0.15.0",
  "description": "AI-powered learning assistant with chat, quizzes, and research tools"
}
```

2. **Build the plugin**:

```bash
npm run build
```

This generates `main.js` from `main.ts`.

3. **Install in Obsidian**:

Copy these 4 files to your Obsidian plugins folder:
- `manifest.json`
- `main.js`
- `styles.css`
- `versions.json`

Folder path: `YourVault/.obsidian/plugins/ljoves-tools/`

4. **Enable the plugin**:
   - Go to Settings â†’ Community plugins
   - Turn off Safe mode
   - Enable "LjoveS Tools"

5. **Configure**:
   - Go to Settings â†’ Study Plugin
   - Enter ONE API key for your preferred provider
   - Select your preferred AI model

### For Production

1. Download the latest release
2. Extract to your Obsidian plugins folder
3. Enable in Obsidian settings

## Usage

### Getting Started

1. **Configure the Plugin** (first time only):
   - Go to Settings â†’ Study Plugin
   - Enter ONE API key (Gemini, OpenAI, or DeepSeek)
   - Select your preferred AI model

2. **Open LjoveS Tools**:
   - Click the brain icon in the ribbon, or
   - Use command palette: "Open LjoveS Tools"

### Using Chat

1. Click "Chat" on the main menu
2. Type your message
3. Tag notes with `[[note name]]` to include context
4. Drag & drop images for vision-capable models
5. Use voice input if configured

### Creating a Quiz

1. Select "Study" â†’ "Quiz"
2. Start typing a note name and select from suggestions
3. Choose question count (5-30)
4. Click "Generate Test"
5. Answer questions with immediate feedback

### Explaining Concepts

1. Select "Study" â†’ "Explain"
2. Select a note or paste content
3. Choose your understanding level
4. Get clear explanations

### Importing Research Papers

1. Select "Research"
2. Enter an arXiv ID (e.g., `1706.03762`) or URL
3. Review paper metadata and token count
4. Customize prompts if desired
5. Generate summary and create a literature note
